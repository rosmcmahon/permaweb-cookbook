# Vue Starter Kits

Vue.js是一种渐进式JavaScript框架，允许构建用户界面。与其他框架不同的是，它在运行时将模板编译为JavaScript，从而得到更小的文件大小和更快的性能。Vue非常适合构建高性能和可扩展的单页面应用程序，因此在前端开发人员中非常受欢迎。
